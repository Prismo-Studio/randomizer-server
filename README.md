# Randomizer Server

Multiplayer server for [Zephyr](https://github.com/Prismo-Studio/Zephyr)'s randomizer feature. Hosts Archipelago multiworld sessions so players can connect and play together.

Built on [Archipelago](https://github.com/ArchipelagoMW/Archipelago) (MIT License).

## Deploy

Designed to run on Fly.io or any Docker host:

```bash
flyctl launch
flyctl deploy
```

Players connect to the server IP on port `38281`.
